ChatGPT chatbot integration for WP.

== Description ==

ChatGPT for WordPress brings the power of OpenAI's GPT-3.5-turbo to your website, allowing you to add a conversational AI chatbot to engage with your visitors. Customize the chatbot's appearance, behavior, and integrate it seamlessly into your WordPress site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/chatbotgpt-wordpress` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Chatbot Settings screen to configure the plugin.

== Frequently Asked Questions ==

= How do I get an API key? =

To use ChatGPT for WordPress, you need an API key from OpenAI. You can obtain one by signing up on the OpenAI website.

= Where do I configure the plugin? =

After activating the plugin, go to Settings -> ChatGPT for WordPress to configure your API key and other settings.

= Can I customize the chatbot's appearance? =

Yes, the plugin allows you to customize various aspects of the chatbot's appearance and behavior from the settings page.
